                              WHAT IS SOULSYNC AI?
================================================================================

SoulSync AI is your personal AI companion.

Think of it like a very smart digital friend who:

  - Remembers every conversation you have with it — forever
  - Understands how you are feeling
  - Stores structured facts about you (name, goals, job, hobbies)
  - Helps you manage your daily tasks automatically
  - Gives you smart advice based on your habits and patterns
  - Talks to you through text (voice mode currently disabled)
  - Supports multiple languages: English, Hindi, and Hinglish
  - Gets smarter and more personal the more you use it
  - Supports multiple users — each person has their own private memory

Unlike regular chatbots (like basic customer service bots),
SoulSync AI is NOT forgetful. It remembers what you told it
yesterday, last week, or even a year ago.

It is like having a personal diary that can talk back to you,
understand you, and help you grow.

================================================================================
                         HOW SOULSYNC AI WORKS — WORKFLOW
================================================================================

Here is the complete journey of what happens when you talk to SoulSync AI.
Explained in simple steps:

--------------------------------------------------------------------------------
STEP 1: YOU SEND A MESSAGE
--------------------------------------------------------------------------------

You open the SoulSync AI app on your browser.
You type something like:

  "I had a really tough day today. I was stressed about work
   and I skipped my gym session."

You can type in English, Hindi (Devanagari script), or Hinglish
(Hindi words in Roman script). SoulSync will automatically detect
your language and respond in the same language.

--------------------------------------------------------------------------------
STEP 2: INTENT IS DETECTED
--------------------------------------------------------------------------------

Before doing anything else, SoulSync classifies what you said
into one of four categories:

  PERSONAL INFO STORE:
    "My name is Rohit" → saves name = Rohit to database
    "My goal is to become a doctor" → saves goal

  PERSONAL INFO QUERY:
    "What is my name?" → instant answer from database (no AI needed)
    "What was the first thing I shared with you?" → retrieves oldest memory

  TASK COMMAND:
    "Remind me to call the doctor tomorrow" → creates task
    "Plan my tomorrow" → creates schedule

  NORMAL CHAT:
    Everything else → goes to AI with memory context

This is called INTENT DETECTION. It makes SoulSync much smarter
than a regular chatbot that treats all messages the same way.

--------------------------------------------------------------------------------
STEP 3: LANGUAGE IS DETECTED
--------------------------------------------------------------------------------

SoulSync automatically detects which language you are using:

  ENGLISH:
    "I had a tough day" → Detected as English
    SoulSync responds in English

  HINDI (Devanagari):
    "मेरा नाम रोहित है" → Detected as Hindi
    SoulSync responds in Hindi (Devanagari script)

  HINGLISH (Hindi in Roman script):
    "Mera naam Rohit hai" → Detected as Hinglish
    SoulSync responds in Hinglish (natural Hindi-English mix)

This happens instantly without any API calls. SoulSync uses pattern
matching to identify Devanagari characters and common Hinglish words.

The language instruction is then added to the AI's system prompt,
ensuring it responds in your preferred language naturally.

--------------------------------------------------------------------------------
STEP 4: YOUR MESSAGE IS UNDERSTOOD AND STORED
--------------------------------------------------------------------------------

SoulSync reads your message and does two things at the same time:

  A) MEMORY STORAGE
     Your message is saved in a database (like a digital diary).
     Every message you ever send is stored safely.

  B) STRUCTURED EXTRACTION
     SoulSync breaks down your message into key pieces:

     Your message: "I was stressed and skipped gym"

     Extracted:
       - Emotion    : stressed
       - Activity   : gym
       - Status     : missed
       - Productivity: low
       - Summary    : User felt stressed and missed gym

     This structured data helps SoulSync understand patterns
     in your life over time.

--------------------------------------------------------------------------------
STEP 4: SOULSYNC SEARCHES YOUR PAST MEMORIES
--------------------------------------------------------------------------------

Before replying, SoulSync searches through ALL your past
conversations to find relevant memories.

This is called RAG — Retrieval-Augmented Generation.

Example:
  You say: "I'm tired again today"

  SoulSync searches and finds:
    - "I felt tired and skipped gym last Tuesday"
    - "Couldn't sleep last night, feeling exhausted"

  Now SoulSync knows this is a PATTERN, not just a one-time thing.

This is what makes SoulSync feel like it truly KNOWS you.

--------------------------------------------------------------------------------
STEP 5: AI GENERATES A PERSONALIZED RESPONSE
--------------------------------------------------------------------------------

SoulSync uses the Groq API (llama-3.3-70b-versatile model) to generate
a response. This is a powerful cloud AI that responds in under 1 second.

The AI automatically responds in the same language you used:
  - If you wrote in English, it responds in English
  - If you wrote in Hindi, it responds in Hindi (Devanagari)
  - If you wrote in Hinglish, it responds in Hinglish

Unlike a normal chatbot, SoulSync's response is enriched with:
  - Your stored personal facts (name, goals, job, etc.)
  - Your past memories retrieved by semantic search
  - Your current situation
  - Language-aware responses that feel natural

So instead of a generic "That's tough!", SoulSync might say:
  "You've mentioned feeling tired a few times this week, Rohit.
   It seems like stress is affecting your sleep and gym routine.
   Have you tried a short 10-minute walk instead on tough days?"

Or in Hinglish:
  "Rohit, tumne is week mein kai baar tired feel kiya hai.
   Lagta hai stress tumhari sleep aur gym routine ko affect kar raha hai.
   Kya tumne tough days pe 10-minute walk try kiya hai?"

--------------------------------------------------------------------------------
STEP 6: TASKS ARE AUTO-DETECTED
--------------------------------------------------------------------------------

If your message contains something that sounds like a task,
SoulSync automatically creates it for you.

Examples:
  You say: "Remind me to call the doctor tomorrow"
  → SoulSync creates: Task "Call the doctor" | Due: Tomorrow

  You say: "I need to finish my project by Friday"
  → SoulSync creates: Task "Finish project" | Due: Friday | Priority: High

You don't have to manually add tasks. SoulSync listens and does it for you.

--------------------------------------------------------------------------------
STEP 7: SMART SUGGESTIONS ARE GENERATED
--------------------------------------------------------------------------------

Over time, SoulSync analyzes your patterns using data science tools
(Pandas and NumPy) and gives you smart advice:

  Pattern detected: You skip gym when stressed (3 times this week)
  Suggestion: "Try a 10-minute walk instead on stressful days"

  Pattern detected: Low productivity on Mondays
  Suggestion: "Start Mondays with your easiest task to build momentum"

  Pattern detected: You feel happy after social activities
  Suggestion: "Schedule more time with friends — it boosts your mood!"

--------------------------------------------------------------------------------
STEP 7: EVERYTHING IS DISPLAYED IN THE APP
--------------------------------------------------------------------------------

The SoulSync app (built with React and Tailwind CSS) shows you:

  LEFT SIDE — Chat Window:
    - Your messages and SoulSync's responses
    - A small note showing how many memories were recalled
    - A note showing if any tasks were auto-created

  RIGHT SIDE — Sidebar with two tabs:

    TAB 1: TASKS
      - All your pending tasks
      - Sorted by priority (High → Medium → Low)
      - You can mark tasks as done or delete them
      - Add new tasks manually

    TAB 2: INSIGHTS
      - Your dominant emotion this week
      - Emotion breakdown chart (happy, stressed, tired, etc.)
      - Smart suggestions based on your patterns

================================================================================
                         THE COMPLETE SYSTEM — ALL PARTS
================================================================================

SoulSync AI is made of 11 modules working together:

  MODULE 1: CORE AI
  -----------------
  The brain of SoulSync.
  Uses Groq API with llama-3.3-70b-versatile model.
  Cloud-based AI that responds in under 1 second.
  Has a built-in persona — knows it is SoulSync AI,
  knows when it was built, and responds warmly.
  Supports multi-language responses (English, Hindi, Hinglish).

  MODULE 2: LANGUAGE DETECTION
  ----------------------------
  Automatically detects the language of user messages.
  Supports: English, Hindi (Devanagari), and Hinglish (Roman script).
  Uses pattern matching for Devanagari characters and common Hinglish words.
  No API calls needed — instant detection.
  Ensures AI responds in the same language as the user.

  MODULE 3: MEMORY SYSTEM
  -----------------------
  Stores every conversation in a PostgreSQL database.
  Like a permanent diary that never forgets.
  Tables stored:
    - users     : who you are
    - memories  : every message ever sent
    - activities: structured data extracted from messages
    - tasks     : all your tasks

  MODULE 4: MEMORY PROCESSING
  ---------------------------
  Reads your raw messages and extracts structured information.
  Detects: emotion, activity, status, productivity level.
  Saves this structured data for pattern analysis later.

  MODULE 5: RETRIEVAL (RAG)
  -------------------------
  Before every response, searches your past memories
  to find the most relevant ones.
  Uses FAISS (a vector search tool by Facebook) to find
  memories that are semantically similar to what you just said.
  This is what makes SoulSync feel like it truly remembers you.

  MODULE 6: SUGGESTION ENGINE
  ---------------------------
  Analyzes your activity data using Pandas (data science tool).
  Detects patterns in your emotions, habits, and productivity.
  Generates personalized suggestions and advice.

  MODULE 7: TASK MODULE
  ---------------------
  Automatically detects tasks from your messages.
  Lets you create, complete, and delete tasks.
  Tasks are sorted by priority (High, Medium, Low).
  Shows due dates and whether the task was auto-created.

  MODULE 8: FRONTEND (User Interface)
  ------------------------------------
  The app you see in your browser.
  Built with React (a popular web framework) and Tailwind CSS
  (for beautiful dark-themed design).
  Features: Chat window, Task panel, Insights panel.

  MODULE 9: AUTHENTICATION
  ------------------------
  Secure user authentication using JWT tokens.
  Password hashing with bcrypt for security.
  7-day token expiration for convenience.
  Each user has their own private memory space.

  MODULE 10: OPTIMIZATION
  -----------------------
  Smart caching system for repeated questions.
  Instant responses for common queries without API calls.
  Direct-answer shortcuts for identity questions.

  MODULE 11: DEPLOYMENT
  ---------------------
  Packages everything into Docker containers.
  Can be deployed on any server or cloud platform.
  Kubernetes configuration for scaling to many users.

================================================================================
                              THE TECHNOLOGY USED
================================================================================

Here is every technology used in SoulSync AI, explained simply:

  PYTHON
  ------
  The main programming language. Like the language SoulSync
  thinks and works in.

  GROQ API
  --------
  Cloud-based AI service that powers SoulSync's responses.
  Uses llama-3.3-70b-versatile model for fast, intelligent responses.
  Responds in under 1 second.

  FASTAPI
  -------
  The web server that handles all requests.
  When you send a message, FastAPI receives it, processes it,
  and sends back the response. Very fast and reliable.

  POSTGRESQL
  ----------
  The database where all memories, tasks, and user data are stored.
  Like a very organized filing cabinet that never loses anything.

  FAISS
  -----
  A tool by Facebook for searching through millions of memories
  very quickly. Used for the RAG memory retrieval system.

  SENTENCE-TRANSFORMERS
  ---------------------
  Converts text into numbers (vectors) so FAISS can search them.
  Like translating words into a language that computers can compare.

  PANDAS & NUMPY
  --------------
  Data science tools used to analyze your activity patterns.
  Like a smart spreadsheet that finds trends in your data.

  REACT
  -----
  The technology used to build the user interface (the app).
  Makes the chat window, task panel, and insights panel.

  TAILWIND CSS
  ------------
  Makes the app look beautiful with a dark, modern design.

  OPENAI WHISPER
  --------------
  Advanced speech recognition model (currently not in use).
  Available for future voice feature implementation.

  DOCKER
  ------
  Packages the entire application into containers.
  Makes it easy to run SoulSync on any computer or server.

  KUBERNETES
  ----------
  Manages multiple containers in production.
  Allows SoulSync to scale to thousands of users.

================================================================================
                           HOW TO START SOULSYNC AI
================================================================================

To run SoulSync AI on your computer, you need two terminals open:

  TERMINAL 1 — Start the Backend (AI Brain):
  -------------------------------------------
  RECOMMENDED — Use the startup script (enforces venv automatically):

    PowerShell:
      powershell -ExecutionPolicy Bypass -File soulsync-ai\start_backend.ps1

    CMD:
      soulsync-ai\start_backend.bat

  The script will:
    1. Check the virtual environment exists at soulsync_env\
    2. Verify all required packages are installed inside it
    3. Start the server using ONLY the venv Python
    4. Refuse to start if venv is missing or packages are absent

  MANUAL (if you prefer):
    Step 1: Activate the virtual environment
            Command: .\soulsync_env\Scripts\activate

    Step 2: Go to the project folder
            Command: cd soulsync-ai

    Step 3: Start the server
            Command: python -m uvicorn backend.main:app --reload --port 8000

  You will see:
    [OK] Venv Python : ...\soulsync_env\Scripts\python.exe
    [OK] All packages verified
    [Schema] All tables created successfully.
    INFO: Application startup complete.

  Backend is running at: http://localhost:8000
  API documentation at:  http://localhost:8000/docs

  IMPORTANT: Never run the backend with system Python.
             Always use soulsync_env. The startup scripts enforce this.

  TERMINAL 2 — Start the Frontend (The App):
  -------------------------------------------
  Step 1: Go to the frontend folder
          Command: cd soulsync-ai\soulsync-frontend

  Step 2: Start the app
          Command: npm run dev

  You will see: "Local: http://localhost:5173"

  OPEN YOUR BROWSER:
  ------------------
  Go to: http://localhost:5173
  You will see the SoulSync AI chat interface.

  OPTIONAL — View API Documentation:
  ------------------------------------
  Go to: http://localhost:8000/docs
  This shows all the available API endpoints with descriptions.
